# AquaSales (React + Vite + Tailwind)

This project splits your single-file React prototype into a proper React app with a clean folder structure.

## Stack
- Vite + React 18
- Tailwind CSS
- lucide-react icons

## Getting Started

```bash
# 1) Install deps
npm install

# 2) Start dev server
npm run dev

# 3) Build for production
npm run build
```
